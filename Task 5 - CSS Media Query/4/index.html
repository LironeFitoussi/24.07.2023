ראשי שאל את הירקן ירקות חדשים הוראות הגעה צור קשר

עגבניה

עגבניה היא ירק טעים ומעושן, מקורו בדרום אמריקה. היא משמשת כמרכיב עיקרי ברוב המנות הים תיכוניות, ותוכנית הבריאות העולמית ממליצה לצרוך אותה יומיומית במנות שונות.

מלפפון

מלפפון הוא ירק ירוק ומרענן שפופולרי בקיץ. עשוי להיבש ולשתות, והוא מספק לנו תחושת קירור מהנה בימי הקיץ החמים. המלפפון מכיל גם רכיבים תזונתיים רבים, כולל ויטמינים ומינרלים.

בצל

הגזר הוא ירק מתוק וצבעוני, הכולל רבים תזונתיים חשובים לגוף. הוא משמש כמרכיב מובהק בסלטים, במנות בשר וכמובן, בגרסאות טבעוניות שונות.

גזר

הבצל הוא ירק ייחודי ובעל טעם חזק ומיוחד. כמו כן, יש לו תפקיד חשוב במטבח כגון בשימוש בכריכים, סלטים, רוטבים ועוד. הבצל קל לשמירה וניתן לאכול אותו טרי או מבושל.

חציל

החציל הוא ירק מסורתי מסדרת הפטרוסיות. בתרבותים רבות, הוא משמש כמרכיב עיקרי במנות כמו בבבה גנוש ומוסקה. החציל ידוע גם בתור חומר גלם עיקרי במנות טבעוניות כמו חמוסיה ופלאפל.
